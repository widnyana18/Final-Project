The front-end is the code that is executed on the client side. This code (typically HTML, CSS, and JavaScript) runs in the user’s browser and creates the user interface.

HyperText Markup Language (HTML) is the core language of almost all Web content. Most of what you see in your browser is a basic description, using HTML.

Cascading Style Sheet merupakan aturan untuk beberapa komponen dalam web sehingga akan lebih terstruktur dan seragam.
